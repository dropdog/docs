### Tmux script

[tmux](https://tmux.github.io/) is a terminal multiplexer. Here are some panes for the project.

  - ls to Drupal folder
  - start docker (or start php server)
  - ```git checkout develop && git pull```
  - open slack url
  - open github repository url
  - open website url(s)
  - open git ui
